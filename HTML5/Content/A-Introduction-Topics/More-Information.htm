1.2 Applying a Custom Style

Applying a custom style can be helpful in creating consistency within a document, and help to make stylizing more streamlined.

  1. Navigate to the ribbon / toolbar.

  2. Select the Home section from the left list of sections.

  3. Select the Styles Pane icon on the right-hand side to open the menu of style options.

  4. Below the current style section, select the button labeled New Style to start creating a custom style.

  5. Title the new style in the text box after Name:, and select modifications under style type, style based on, style for following paragraph, and the formatting options.

  6. As modifications are selected, the preview window will update with text reflective of the chosen style.

  7. Select OK to save the style to the options in the style pane.

  8. To apply the style, select the text that will be modified by highlighting the text, and select the name of the custom style on the right-hand list of styles in the style pane option window.
